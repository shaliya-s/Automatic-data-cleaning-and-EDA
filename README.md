Data CleanSight – Automated Data Cleaning and Exploratory Data Analysis Platform

Project Overview

Data CleanSight is a Flask-based web application developed to automate data cleaning, preprocessing, exploratory data analysis (EDA), and visualization for structured datasets.

The platform helps users transform raw datasets into analysis-ready data by automatically handling common preprocessing tasks such as missing value treatment, duplicate removal, outlier detection, and dataset exploration.

This project aims to reduce manual preprocessing effort and improve the efficiency of data preparation for analytics and machine learning workflows.



Project Objective

- Automate repetitive data cleaning tasks.
- Simplify exploratory data analysis.
- Generate meaningful visualizations automatically.
- Provide a user-friendly interface for dataset preprocessing.
- Support machine learning workflows through clean and reliable data preparation.

Project Workflow

1. User uploads a CSV dataset.
2. Dataset is validated by the backend.
3. Missing values are identified and handled.
4. Duplicate records are removed.
5. Outliers are detected.
6. Dataset statistics are generated.
7. Visualizations are created automatically.
8. Cleaned dataset is made available for download.
